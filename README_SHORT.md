# Ember

Приложение с ежедневными цитатами на Flutter (iOS и Android). Каждый день одна
карточка-мысль; отмечаешь прочитанной — растёт стрик. Прочитанное копится в архиве.

## Функционал по заданию

- Онбординг из двух экранов с кнопкой «Продолжить».
- Paywall на входе: месяц и год (год со скидкой), «Продолжить» эмулирует покупку.
- Главный экран: цитата дня и архив прочитанного.
- Подписка хранится в SharedPreferences: после покупки paywall на следующем запуске
  пропускается.

## Сверх задания

100 цитат по четырём темам со своими шрифтами, ежедневные напоминания с выбором
времени, покупка подписки в любой момент, анимация огонька.

## Архитектура

Один `AppState` (ChangeNotifier) как источник правды, только он работает с хранилищем.
Стартовый маршрут выбирается в одном методе. Логика стрика — отдельная чистая функция
под тестами.

```
lib/  core · data · state · features · widgets
```

## Запуск

```bash
flutter pub get
flutter run
```
