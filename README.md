# Ember

Приложение с ежедневными цитатами. Каждый день показывается одна карточка-мысль;
когда отмечаешь её прочитанной, растёт счётчик дней подряд (стрик). Прочитанные
карточки складываются в архив.

Тестовое задание на Flutter. Цель платформы — iOS и Android, контент полностью
локальный, интернет не нужен.

## Функционал по заданию

1. Онбординг из двух экранов (короткий смысл приложения и выбор темы), кнопка
   «Продолжить».
2. Paywall на входе: два плана, месяц и год (год со скидкой и выбран по умолчанию).
   Кнопка «Продолжить» эмулирует покупку, реального биллинга нет.
3. Главный экран: цитата дня, кнопка «Прочитал», архив прочитанного, избранное.
4. Состояние подписки хранится в SharedPreferences. Если подписка куплена, при
   следующем запуске paywall пропускается и сразу открывается главный экран.

## Что добавил помимо задания

- Подписку можно оформить не только на входе, но и позже, с баннера на главном.
- 100 цитат по четырём темам (спокойствие, фокус, мотивация, стоицизм). У каждой
  темы свой шрифт и акцентный цвет.
- Ежедневное локальное уведомление-напоминание с выбором времени.
- Анимированная иконка огонька и небольшая анимация при росте стрика.

## Архитектура

Состояние держит один `AppState` (ChangeNotifier) — это единственное место, которое
обращается к хранилищу через `PrefsService`. Экраны и виджеты данные из хранилища
напрямую не читают.

Выбор стартового экрана собран в одном методе `AppState.resolveInitialRoute()`: нет
онбординга — показываем онбординг; нет подписки — paywall; иначе главный экран.

Подсчёт стрика вынесен в отдельную чистую функцию (`state/streak.dart`) без зависимости
от Flutter, поэтому он покрыт юнит-тестами. Правило простое: тот же день — без
изменений, вчера — плюс один, пропуск — сброс на единицу (сравниваются только даты в
формате `yyyy-MM-dd`).

Ключи в SharedPreferences объявлены константами в `core/constants.dart`:
`onboarding_done`, `is_premium`, `streak_count`, `last_open_date`, `favorites`,
`selected_theme`, `reminder_enabled`, `reminder_hour`, `read_quotes`.

```
lib/
  main.dart, app.dart
  core/      constants.dart, theme.dart, mood_style.dart
```

(+ timezone, flutter_timezone).

## Что доделал бы с большим запасом времени

- Реальные покупки через `in_app_purchase` за тем же флагом `is_premium`.
- Заморозку стрика как настоящую функцию подписки.
- Доступ к хранилищу через интерфейс-репозиторий, чтобы проще было тестировать и
  при желании подключить бэкенд.
- Генерацию цитаты под выбранную тему через LLM с откатом на локальный JSON.
